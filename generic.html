Multi-Functional Quadruped Robot with Advance Locomotion, Remote Operability and Cognitive Capabilities

Quadruped robots have become more and more common in recent times as they have shown significant advantages compared to wheeled robots in certain aspects. The technologies regarding the quadruped robots have seen a vast improvement in the last decade. These quadruped robots have many real-life applications in the field of monitoring, surveillance, mapping, advance locomotion and many different operations. Although, consumer level quadruped robots have been in the market for a few years now, there are certain things lacking in the current market. These lacking include expensive cost, hard to source components and high-power consumption. Our proposed project aims to mitigate the current limitations of the quadruped robots. This project introduces the conceptualization and execution of a multifunctional quadrupedal robot, prioritizing the attainment of sophisticated locomotion capabilities, remote operability, and cognitive capabilities. The robotic system incorporates a Raspberry Pi as its primary computational unit, accompanied by an Arduino Mega and an Arduino Uno for motor regulation and other functionalities. The implementation of twin gear cycloidal drives, driven by NEMA 17 stepper motors, in conjunction with inverse kinematics algorithms, allows for accurate leg motions. This enables the robot to achieve forward, backward, and rotating locomotion, while also avoiding obstacles and generating 2D maps of its environment. Significantly, the robot possesses the capacity to remotely view its real-time video feed via TeamViewer. Additionally, it is equipped with object detection capabilities through the utilization of OpenCV and TensorFlow Lite. This enables the robot to effectively identify and promptly alert the presence of persons. The Python graphical user interface (GUI) application facilitates manual camera control, locomotion management, and the ability to toggle human detection. By incorporating the integration of a third-party GPS Application Programming Interface (API) along with a hardware GPS Module for the purpose of generating coordinates, this project expands its potential use in surveillance, monitoring and search and rescue. By the end of the project completion, we have made a functional quadruped robot that is both affordable with components that can be easily sourced and has low power consumption with sustainable 3D printed parts.
